# AndroidToolKit
Here are some android examples.

+ Location-1-Kotlin: 

	按下按鈕，呈現使用者上次定位的座標，如果使用者手機沒有任何紀錄，可能為空。

+ Location-1-Java: 同上(還沒改)。

+ Location-2-Kotlin: 

	按下按鈕，呈現使用者上次定位的座標，並可以啟動或停止更新座標。

+ Location-3-Kotlin:

	按下按鈕，呈現使用者當前的座標，但會不會很耗電還不知道。
	
+ Permission-1-Kotlin: 

	開啟兩個以上危險權限的方法。

+ ScanQrCode-1-Kotlin: 

	掃描 QR-code 以及切換 Activity 的細節。
